# AuxiliarPixelmon
EN A Python program built with LLMs and LangGraph, using a graph-and-node structure, where the user provides the opponent’s team in Pixelmon/Pokémon. The software analyzes it, builds the optimal team to defeat the opponent, and explains the reasoning behind each choice, including which Pokémon to use against specific threats.

🇧🇷 Um programa em Python feito com LLM e LangGraph com sua estrutura de grafos e nós, onde o usuário informa a equipe que vai enfrentar no Pixelmon/Pokemon, e o software analisa e monta a equipe perfeita para ganhar do adversário, além de explicar todo o porque de cada escolha e quais Pokémon usar contra outro.
